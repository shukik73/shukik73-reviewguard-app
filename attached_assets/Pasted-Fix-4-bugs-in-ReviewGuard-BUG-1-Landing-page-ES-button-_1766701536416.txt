## BUG 2: Login modal doesn't close when clicking outside

The login modal stays open when clicking outside of it. Add click-outside-to-close.

### Find the login modal wrapper div (the overlay/backdrop) and add onclick:

Change from something like:
```html
<div class="fixed inset-0 bg-black/50 flex items-center justify-center z-50" id="login-modal">
```

To:
```html
<div class="fixed inset-0 bg-black/50 flex items-center justify-center z-50" id="login-modal" onclick="if(event.target === this) closeLoginModal()">
```

### Or if using a separate backdrop div, add onclick to the backdrop:
```html
<div class="absolute inset-0 bg-black/50" onclick="closeLoginModal()"></div>
```

### Make sure closeLoginModal() function exists:
```javascript
function closeLoginModal() {
  const modal = document.getElementById('login-modal');
  if (modal) modal.classList.add('hidden');
}
```

### Also add a close button (X) to the login modal for accessibility:
```html
<button onclick="closeLoginModal()" class="absolute top-4 right-4 text-gray-400 hover:text-gray-600">
  <svg class="w-6 h-6" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M6 18L18 6M6 6l12 12" />
  </svg>
</button>
```

## BUG 4: Feedback inbox error message appears behind modal

The error toast/notification has a lower z-index than the response modal.

### Find the toast/error notification CSS and increase z-index:

Look for toast styles (might be in CSS or inline) and change:
```css
.toast, .notification, .error-message, [data-toast] {
  z-index: 9999 !important;
}
```

### Or if it's a specific element, find it and add higher z-index:
```html
<div id="toast" class="fixed ... z-[9999]">
```

The feedback response modal is likely z-50 (z-index: 50), so the toast needs to be higher like z-[9999] or z-[100].

### If using a toast library or custom toast function, update the z-index there:
```javascript
function showToast(message, type) {
  const toast = document.createElement('div');
  toast.className = '... z-[9999]';  // Make sure z-index is very high
  // ...
}
```
