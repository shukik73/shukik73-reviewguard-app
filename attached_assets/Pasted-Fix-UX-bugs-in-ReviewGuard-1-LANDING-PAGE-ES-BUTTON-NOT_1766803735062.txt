## 1. LANDING PAGE ES BUTTON NOT TRANSLATING

Find the language toggle buttons on the landing page and ensure they call the translation function:
```javascript
// Make sure these buttons exist and work:
<button onclick="setLang('en')" class="...">EN</button>
<button onclick="setLang('es')" class="...">ES</button>

// And setLang function triggers translation:
function setLang(lang) {
  localStorage.setItem('lang', lang);
  if (typeof autoTranslate !== 'undefined') {
    autoTranslate.setLang(lang);
  }
  if (typeof landingTranslate !== 'undefined') {
    landingTranslate.translate();
  }
  // Update button states
  document.querySelectorAll('[data-lang-btn]').forEach(btn => {
    btn.classList.toggle('bg-purple-600', btn.dataset.langBtn === lang);
    btn.classList.toggle('text-white', btn.dataset.langBtn === lang);
  });
}
```

Also ensure landingTranslate.js is loaded on the landing page.
