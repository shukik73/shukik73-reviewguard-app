## 3. ADD EVENT TRACKING

Find these actions in the JavaScript and add posthog.capture() calls:

### After successful SMS send:
```javascript
posthog.capture('sms_sent', { 
  vertical: localStorage.getItem('vertical') || 'repair'
});
```

### After QR Stand printed:
```javascript
posthog.capture('qr_printed');
```

### After audit form submitted:
```javascript
posthog.capture('audit_lead_captured');
```

### After waitlist/ServiceTitan modal submitted:
```javascript
posthog.capture('waitlist_joined', { feature: 'servicetitan' });
```

### After language toggle:
```javascript
posthog.capture('language_changed', { lang: newLang });
```

### After successful login (in the login success handler):
```javascript
posthog.identify(user.email, {
  name: user.businessName,
  email: user.email,
  plan: user.plan || 'free_trial'
});
```
